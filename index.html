Windows Checklist

Local Users and Groups

  • Disable accounts for all unauthorized users
  • Administrators group should only contain authorized administrators
  • Guests group should only have Guest account in it

Lockout Policy

  • Account lockout duration: 30 minutes
  • Account lockout threshhold: 10 incorrect passwords
  • Reset account lockout counter after: 30 minutes

Password Policy

  • Enforce passowrd history: 5 passwords rememebred
  • Maximum passowrd age: (30-90 days)
  • Minimum passowrd age: (10-30 days)
  • Password must meet the complexity requirements: Enabled
  • Store password using reversible encription: Disabled
